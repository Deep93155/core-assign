list of tags in html:

  1. "base tag"

    2. Specifies the base URL/target for all relative URLs in a document


  2. "a" tag

    3. Defines a hyperlink


  3. "b" tag

    4. Defines bold text


  4. "body" tag

    5. Defines the document's body


  5. br tag

    6. Defines a single line break


  6. buton tag

    7. Defines a clickable button


  7. audio tag

    8. Defines embedded sound content


  8. DOCTYPE tag

    9. Defines the document type


  9. comment tag

    10. Defines a comment


  10. form tag

    11. Defines an HTML form for user input
